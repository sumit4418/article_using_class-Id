Difference between inline,Embedded and External CSS
Inline Embedded External
An inline style rule is coded directly into an HTML tag within the flow of your source code, just as an HTML attribute is used. Embedded styles are set in the HEAD section of your web page. Embedded styles are set in the HEAD section of your web page.
They’re useful for quick and permanent changes but are less flexible than external and internal stylesheets, as each inline style you create must be edited separately if you’re making a design change.
